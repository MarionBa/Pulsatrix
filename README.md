#### **g2 functions codes:**



There are two different type of codes:

1- the 3kHzData\_g2 code process .seq files

2- the line camera codes which process the .tiff images



The line camera data was used to process 25, 30 and 50kHz sampling videos. Eventually the 50kHz data was the most

useful, hence the 50kHz\_...py codes in the repository.
